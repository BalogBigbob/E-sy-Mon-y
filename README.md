⛏️ E-sy Mon-y – Crypto Mining in Everyone's Pocket
E-sy Mon-y is an Android-based Monero (XMR) mining application designed to make cryptocurrency production accessible to anyone without expensive hardware or technical expertise. The software is built with safety and device longevity as top priorities.

🚀 Mission: Democratizing Mining
We believe that entering the world of crypto should not be tied to multi-million forint hardware investments.

Accessible to Anyone: All you need is your existing smartphone. No risk, no heavy investment.
Simplicity: Start mining with just a few clicks while the app works seamlessly in the background.
Community Power: By using the same wallet address on multiple phones within a family or group, small computing capacities add up to produce visible results.
📅 The Unrivaled Practical Calendar Feature
The heart of the application is its dual-purpose, highly practical calendar:

Mining Log: Automatically records and displays daily hashrate averages and time spent mining. This allows you to track your devices' performance precisely, broken down by day.
Personal Organizer: Beyond mining data, it serves as a full-featured calendar and reminder tool. You can record important tasks and appointments, and the app will provide notifications for them.
Stats & Lifestyle: This is the only mining app that fuses passive profit generation with daily organization.
✨ Key Technical Features
Intelligent PWM Throttling: Mobile processors are primarily optimized for burst performance. Our app's PWM logic follows this behavior: it mines in short, intensive bursts to prevent long-term overheating and battery degradation.
Core Management: Selectable thread and core counts (support for Efficiency and Performance cores).
Stability: A rock-solid background service that continues to produce even after the app is closed.
🛡️ Hardware Protection & Security
Target Temperature: Automatic throttling above a set limit (e.g., 55-63°C).
Battery Protection: Selectable charge level (e.g., 30%) below which the mining process stops.
Power Limit: Customizable Watt limit to keep processor load under control.
📖 Documentation & Help
Detailed user guides and technical descriptions are built directly into the application. To access them, click the red "M" (Monero) icon in the top-left corner of the main screen.
